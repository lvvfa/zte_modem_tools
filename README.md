# zte_factroymode.py
系列通用密码实验
```
python3 zte_factroymode.py telnet
```
指定密码端口开telnet
```
python zte_factroymode.py --user CUAdmin --pass Aa888888 --ip 192.168.1.1 --port 80 telnet open
```

$ python3 ./zte_factroymode.py -h
